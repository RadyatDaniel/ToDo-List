# 📝 ToDo List App

A full-stack ToDo application built with **FastAPI (backend)** and **React (frontend)**.  
It allows users to create, update, and delete tasks while persisting data in a PostgreSQL database.

---

## 🚀 Features
- Add, update, and delete tasks
- PostgreSQL database integration
- RESTful API built with FastAPI
- Frontend built with React
- Connected frontend and backend

---

## 🛠️ Tech Stack
- **Backend:** FastAPI, SQLAlchemy, PostgreSQL
- **Frontend:** React, JavaScript, CSS
- **Database:** PostgreSQL
- **Deployment:** GitHub (future: Render/Vercel)
